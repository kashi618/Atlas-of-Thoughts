## Faster Character Data to Character Variable
- getchar()
- putchar()

``` c
#include <stdio.h>

int main()
{
	 char my_char = ' ';
    
	 printf("Enter any character\n");
	
	// Takes an input for the user
	my_char = getchar();
	
    // Prints out character
    putchar(my_char);
    
    // This is the long way to print it out
    printf("%c",my_char);
    
	return 0;
}

```
