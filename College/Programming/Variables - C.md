- Piece of memory in a program used to store data
- Variables are stored in RAM

## Declaring Variables
```c
variableType variableName;
```

### Example
``` c
int var1;
```

## Initializing Variables
```c
variableType variableName = value;
```

### Example
``` c
int var1 = 100;
```
