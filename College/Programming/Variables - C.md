## Initializing Variables
``` c
#include <stdio.h>

int main() {
	// Initializing and assigning values to variables
	int int1;
	int int2 = 10;
	
	float float1;
	float float2 = 12.321;
}
```
## Rules for Naming Variables
- Must **start** with a letter or an underscore
