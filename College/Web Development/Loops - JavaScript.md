## For Loop
Similar to 
``` js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```
```
OUTPUT:
0
1
2
3
4
```

## While Loop
``` js
i = 0;
while (i < 5) {
	console.log("hii");
	i++;
}
```
```
OUTPUT:
hii
hii
hii
hii
hii
```
