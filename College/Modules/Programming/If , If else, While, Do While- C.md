## If \_\_\_ 
Used for comparing
```c showlinenumbers
if (x>10) {
	// Run code
}
```

### If \_\_\_ Else \_\_\_
Use for comparing. If it is false, then do something else
```c showlinenumbers
if (x>10) {
	// Run code
}
else if (x==5) {
	// Run Code
}
else {
	// Run Code
}
```

### Nested If Statement
This is when you have an if statement inside of an if statement
```c showlinenumbers
if (x > 10) {
	if (y > 10) {
		// Run code
	}
}
```
