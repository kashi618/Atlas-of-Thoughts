## Exiting
`:q`
OR (force quit)
`:q!` 

## Insert Mode
This is used for typing text
`i`

### Inset Mode and Add New Line
This goes into insert mode, and adds a line underneath it
`o`
## Edit Mode
This is used to edit the test
`esc`

## Saving a File
`:w` 
OR (Save with file name)
`:w textName.type`
OR (Save and quit)
`:wq`
