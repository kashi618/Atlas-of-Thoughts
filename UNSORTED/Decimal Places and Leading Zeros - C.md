## Decimal Place
Prints out a variable, with decimal place
- Does not affect value stored in variable, only the output
``` c
printf(".2f")
```

## Leading Zeros
Makes a number have a specific amount of digits before the zero.
- Does not affect value stored in variable, only the output
```c
printf("")
```
