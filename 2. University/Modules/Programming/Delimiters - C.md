In order to print out or store data inside of variables, a delimiter is needed to specify the **type of data** you want to input/output.

| Delimeter | Type                                  |
| --------- | ------------------------------------- |
| %d        | Integer                               |
| %f        | Float                                 |
| %c        | Character                             |
| %p        | [[Pointers - C]]                      |
| %s        | [[Strings - C]] |
