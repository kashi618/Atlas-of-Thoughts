## View Current Permissoins
```bash
ls -l
```

## Types of Permissions
**Read Access**
`r  or  r-`
- Allows the user to seek the contents of a file

**Write Access**
`w  or  w-`
- Allows the user to change and modify the contents of a file

**Execute Access**
`x  or  x-`
- Allows the user to execute/run a file
