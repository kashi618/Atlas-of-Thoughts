## Types of Inserts
**Full Insert**
This inserts a value for every column in the table

**Partial Insert**
This inserts some values, accepting default values for the other columns

## How To Add a New Row
```sql
INSERT INTO tableName (columnName, ...) VALUES (colValue, ...);
```
