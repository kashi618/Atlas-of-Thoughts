## main(int argc, char* *argv)
`argc` is a variable containing the amount of command line arguments
`argv` is a character pointer array containing all the command line arguments
 - `argv[0] -> first command line argument`
 - `argv[1] -> second command line arguments, etc`

**Example**
`cp file1 file 2`

`argv[0]` is `cp`
`argv[2]` is `file2`

`argc[]` is 3
