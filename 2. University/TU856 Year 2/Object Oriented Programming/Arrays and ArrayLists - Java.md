## Differences?
**Arrays**
- Stores primitive types and reference types (object, as they are a memory address)
- Size is set at 

**ArrayList**
- Stores only objects of the same type
- Size is flexible, and can be extended

An arrayList is similar to an array, but with a flexible size.
- Size flexible
- Can store any type of object
