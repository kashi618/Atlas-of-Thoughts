### B)
To push a node onto a stack, you will need to first:
Create a new stack node in heap memory `StackNode newNode = malloc(sizeof(StackNode));`
Then, give the node a value. `newNode->data = value;`
Lastly we set the topPtr of the stack, to point to the address of the new node. `*topPtr = newNode;`
