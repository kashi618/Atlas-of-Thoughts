## Adaptive
These methodology is change driven. It focuses on constant refactoring and code improvement. It starts by creating a small idea, and then iterating it as the requirements are found. This allows the stakeholders to see the program through its iterations, allowing for quick changes if need

- [[Agile Development]]

## Predictive
This methodology is plan driven. It focuses on defining and planning all the requirements at the beginning, and then building the final product.
