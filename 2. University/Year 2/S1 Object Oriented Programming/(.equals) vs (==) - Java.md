## a == b
Checks if two items are the exact same
- Used to check values if used on a primitive data type
- Used to check memory addresses if used on a class

**Example**
```java
String s1 = "barnie";
String s2 = "barnie";
if (s1 == s2) {
	System.out.println("equivalent !");
}
```
They will never be the same, because you are comparing the memory address of `s1` with `s2`, not the contents


## a.equals(b)
Checks if the contents of two classes or objects are the same

**Example**
```java
String s1 = "barnie";
String s2 = "barnie";
if (s1.equals(s2)) {
	System.out.println("equivalent !");
}
```
This will be true, because `.equals()` will compare the contents of `s1 (barnie)` with `s2 (barnie)`
