## Subscript Notation
`array_name[i]`

"Square bracket notation" - Michael Collins
When \[ and \] are used to access the elements of an array


## Pointer Notation
`*(array_name + i)`

When you use the [[Dereference Operators - C|dereference]] operator to access the array
