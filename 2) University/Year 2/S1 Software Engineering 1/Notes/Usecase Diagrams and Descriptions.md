## Description for a Library System
**Return Book:**
1. Library system prompts user for library card
2. User scans library card
3. Library system prompts success
4. User returns book into return bin

**Borrow Book**
1. Library system prompts user for library card and books to borrow
2. User scans library card and books
3. Library system indicates success

**Borrow Book and Pay Fine**
Extends Borrow Book on line 3
4. a) System prompts user to pay existing fines using card
5. a) User accepts and enters card information
6. a) Library system accepts payment
7. a) Library system indicates success
