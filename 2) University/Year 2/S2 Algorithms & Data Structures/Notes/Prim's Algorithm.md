## Prim's Algorithms
Greedy algorithm that focuses on building a single tree, from vertices. Starting from an arbitrary vertex, it finds and connects to a non-existing vertex with the lowest value edge, and repeatedly adds the lowest value edge to any vertex already in the tree, one at a time. Any edge is valid, unless it connects to existing vertexes togethor. (because it causes a loop)

- Starts from an arbitrary vertex
- From that vertex, connect the edge of the lowest value
- Then, find the next edge that is lowest, that does NOT connect to an existing vertex
- This is done until all vertexes are connected
