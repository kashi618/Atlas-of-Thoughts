### Graph Algorithms
There are a variety of useful algorithms, that are used to operate on a graph data structure, such as:
- Depth first traversal 
- Bread first traversal
- Best first traversal
- Shortest path tree (3 algorithms, dijkstra)
- Minimum spanning tree (2 algorithms)
- Hamiltonian cycle
- Eulerian cycle
