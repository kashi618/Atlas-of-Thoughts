## Traveling Sales Person
Given a list of cities and the distances between each city, find the shortest possible route that goes to each city exactly once, and returns to the starting city.
- Note: it is an intractable problem, meaning it is not feasible to find the absolute shortest path. Instead, we focus on a mangeable "good enough" path
