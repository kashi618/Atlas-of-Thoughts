## Minimum Spanning Trees
Given a connected weighted undirected graph, a spanning tree is a subgraph that:
- Contains all vertices of the original graph
- Is a tree, that is connected and contains no cycles/loops
- Has the minimum possible total edge weight among all spanning trees of the graph
