## Mac Addresses
To identify a device on a LAN, a unique 48-bit hardware address is assigned to each device.
- They are also included in frame headers, to identify the sender and receiver of a frame
- The NIC, also uses these addresses to filter frames, so that the CPU is only interrupted for frames addressed directly to it

### Assigning Addresses
There are three main ways to assign a MAC address
1. **Static**: The address is set by the manufacturer
2. **Dynamic**: The address is assigned when the address boots up
3. **Configurable**: The address is set by an administrator
