## Types
Each type of communication, requires its own address

### Unicast
This is station to station communication
It requires the use of unique MAC addresses
- All [[Network Interface Card (NIC)|NICs]] are configured with a unique MAC address


### Broadcast
This is station to some station communications

### Multicast
This is station to all stations communications
