## Software Testing
Testing in software, is used to show that a program does what it is designed to do, discovering program defects before it is put into use.

There are two ways to test a program:
1. Validation Testing
2. Defect Testing

## Phases of Testing
Commercial software goes through three stages of testing:
1. Development Testing
2. Release Testing
3. User Testing

### Development Testing
**When**
During the development phase

**Who**
Developers and QA engineers

**Goal**
Used catch bugs and defects early, and to verify code functionality

### Release Testing
**When**
Right before product release/deployment

**Who**
Separate QA team

**Goal**
Ensures that the program is ready for users, and meets the requirements of the stakeholders.


### User Testing
**When**
Before or after release

**Who**
Real and potential users of the product

**Goal**
Verifies the real world effectiveness of the product. Testing is done in the environment of the users/potential users.
