## Logical Operators (and, or, not) - C

| Logical Operator | Meaning |
| ---------------- | ------- |
| &&               | AND     |
| \|\|             | OR      |
| !                | NOT     |
### Example
``` c
int a = 1;
int b = 2;

if (a == 1 && b == 2) {
	printf("A is 1, and B is 2)
}
```
